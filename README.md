# 📊 Transaction Analyzer (TDA) - Java

An efficient financial transaction monitoring application written in Java. This project demonstrates core object-oriented programming practices, manual array population control, and graphic interactions using standard Java libraries.

## 🛠️ Technical Details
* **Custom Dynamic Control:** Designed specifically around native arrays (`double[]`) avoiding high-level dynamic collections to focus on strict memory layout boundaries.
* **Data Flow:** Features input capacity limits, active element tracking using manual indices, and algorithmic calculations for total accumulation and arithmetic averages.
* **GUI Component:** Utilizes `javax.swing.JOptionPane` to manage interactive window loops and generate clean graphical summaries.

## 🚀 Execution Structure
* `TransactionAnalyzer.java`: The core model holding the encapsulated array logic, memory bounds, and math methods.
* `Main.java`: Contains the interactive execution loop control flow (`do-while`) managing real-time entries and constraints.
